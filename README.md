# timePicker 
## Time picker v-0.1.2

Useg:
### CDN:
```html
<script src="https://cdn.jsdelivr.net/gh/Qasim345/timePicker@master/dist/time-picker.min.js"></script>
```
### locale:
```html
<script src="dist/time-picker.min.js"></script>
```
## Example:
```html
<script type="text/javascript">
timePicker("#myInput",{
  bg: "light",
  shadow: true,
  lang: "en",
  format: 12
})
</script>
```
